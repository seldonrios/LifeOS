# Simulation

## Purpose

Describe the bounded simulation capability in the Phase 1 reference architecture and how it can expand later.

Simulation in LifeOS means evaluating possible future paths before a user commits time, money, or effort.

## Example Questions

- Should I start a micro farm?
- Should I invest in new equipment?
- What happens if I shift time from one project to another?

## Relation to Phase 1

Phase 1 can support bounded local simulation using Python services, Monte Carlo methods, or simple agent-based models over life-graph and production data.

It should not be treated as a complete universal forecasting engine. It is a decision-support tool built on local context.
