# LifeOS Documentation

## Purpose

Provide the canonical index for the LifeOS docs set and give contributors a clear reading path through the Phase 1 vision and architecture.

## Active MVP Surface

Current executable MVP targets are:

- `@lifeos/cli`
- `@lifeos/goal-engine`
- `@lifeos/life-graph`
- `@lifeos/event-bus`
- `@lifeos/module-loader`
- Runtime modules: `@lifeos/reminder-module`, `@lifeos/calendar-module`, `@lifeos/scheduler-module`, `@lifeos/research-module`, `@lifeos/notes-module`, `@lifeos/weather-module`, `@lifeos/news-module`, `@lifeos/personality`, `@lifeos/orchestrator`, `@lifeos/sync-core`
