# Deployment Support Matrix

| OS | Dev Container | Native Install | Docker-only Runtime |
| --- | --- | --- | --- |
| macOS | Supported (recommended) | Supported (advanced) | Supported |
| Windows | Supported (recommended) | Supported (advanced) | Supported |
| Ubuntu | Supported (recommended) | Supported (advanced) | Supported |

Notes:

- Dev Container is the default supported contributor path.
- Native install is for advanced workflows and local tuning.
- Docker-only runtime is supported for service-oriented deployments.
