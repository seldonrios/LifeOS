# 🚀 Phase 1 MVP Complete – Single-Node Sovereign Personal AI Node (v0.1)

**Status**: Shipped ✅

LifeOS v0.1 is now live:

- `lifeos goal "..."` -> structured Life Graph plan
- `lifeos status` -> dashboard summary
- `lifeos task list|complete|next` -> task execution loop
- `lifeos tick` -> reactive overdue detection
- `lifeos events listen` -> live event stream (NATS or in-memory fallback)
- `lifeos modules` -> pluggable module runtime (includes reminder module)

## Try It In <5 Minutes

```bash
pnpm install
pnpm lifeos demo
```
