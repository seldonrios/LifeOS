# LifeOS v0.2.0 is here - Help build the ecosystem

LifeOS has reached `v0.2.0` with local voice control, contextual memory, proactive orchestration, and multi-device sync.

Now we are focused on growing the module ecosystem.

## What shipped

- Voice core with wake flow and local STT/TTS pipeline
- Daily assistant modules: research, notes, weather, news
- Persistent memory + personality profile
- Proactive orchestrator and daily briefing
- Local-first device sync over event deltas

## Updated roadmap

- Phase 6: public release, contributor onboarding, and module ecosystem growth
- Phase 7: deep integrations (smart home, productivity suites, wearables)
- Phase 8: explainability and trust tooling (decision traces, memory audit UX)

## First community modules wanted

1. Smart Home bridge
2. Email assistant
3. Fitness tracker integration

## Contribution target

We want at least one high-quality community module PR merged within a week of this issue going live.

## Definition of done for new modules

- Implements `LifeOSModule`
- Subscribes/publishes well-scoped events
- Persists meaningful state into Life Graph
- Includes tests for success + degraded paths
- Documents setup and commands in a package README
