# LifeOS v0.2.0 Launch Thread Draft

## Post 1

LifeOS `v0.2.0` is live.

This release makes local-first personal AI practical:

- voice interface
- contextual memory
- proactive orchestration
- multi-device sync

All private. All local. No vendor lock-in.

## Post 2

What shipped:

- "Hey LifeOS" voice flow
- Research, Notes, Weather, News modules
- Personality preferences and daily briefing
- Sync across devices via local event deltas

## Post 3

Try it:

```bash
git clone https://github.com/seldonrios/LifeOS
cd LifeOS
pnpm install
pnpm lifeos voice consent
pnpm lifeos voice start
```

Then say:

`Hey LifeOS, give me my daily briefing`
